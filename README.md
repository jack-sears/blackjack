# Blackjack Trainer Project  
### By Jack Sears - 2025

Welcome to my BlackJack Trainer! This program simulates a Blackjack game and provides feedback on optimal strategies.
Optimal strategies can be found under the PDF Guide section listed below.

You can download the program [here](https://github.com/jack-sears/blackjack/blob/main/dist/21-v1.exe). The program currently just runs in the terminal.

## How to Use

### GUI Version (Recommended)
1. Install dependencies: `pip install -r requirements.txt`
2. Run the GUI version: `python 21-v1-gui.py`
3. Click buttons to play - see actual cards and visual feedback!

### Terminal Version
1. Clone or download this repository.
2. Open a terminal in the project directory.
3. Run the program using `python 21-v1.py`.
4. Follow the prompts to play the game!

## Features
- **Visual Cards**: See actual card graphics in the GUI version
- **Strategy Feedback**: Get instant feedback on whether your moves match optimal basic strategy
- **Statistics Tracking**: Track wins, losses, draws, and accuracy
- **Split Support**: Practice splitting pairs with visual feedback
